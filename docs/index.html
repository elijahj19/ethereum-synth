This is my MUSC-075 Final Project.This web app communicates with an Ethereum smart contract to change certain parameters on a Teensy 3.2 that algorithmically produces sounds.

Ensure you have the Metamask extension on Chrome installed. Click the "Connect to Contract" button to start.

Change the parameters as you want. When you are done, press "Submit Changes". If you want to see what parameters are currently being used click "Reset".
